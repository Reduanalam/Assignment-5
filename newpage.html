Question-1: What are the different ways to select an element in the DOM?

Answer:
1. document.getElementById("id") → Selects an element by ID.
2. document.getElementsByClassName("class") → Selects elements by class (returns a collection).
3. document.getElementsByTagName("tag") → Selects elements by tag name.
4.document.querySelector("selector") → Selects the first matching element.
5. document.querySelectorAll("selector") → Selects all matching elements (returns a NodeList).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:
1.innerHTML → Returns or sets HTML content, including tags.
2.innerText → Returns or sets visible text, ignoring hidden.
3.textContent → Returns or sets all text, including hidden elements, but without parsing HTML.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique where a parent element handles events for its child elements
using event bubbling. It improves performance by reducing event listeners.

Question-4: What is event bubbling in the DOM?

Answer: Event bubbling is when an event triggered on a child element propagates up to its parent elements.
It allows handling events at a higher level.
